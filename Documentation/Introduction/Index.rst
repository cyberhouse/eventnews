.. _what-it-does:

What does it do?
================

This extension extends :t3ext:`news` with additional event functionality.
This includes additional fields and an extra view to display a calendar.

.. _requirements:

Requirements
============

The requirements for this extension are:

- EXT:news 3.1.0+
- TYPO3 CMS 7.6 LTS
